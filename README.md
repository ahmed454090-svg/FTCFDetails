# 🚀 AI Content Studio Pro (Utility Web App)

An elegant, lightweight AI-powered content generator built with **HTML, Tailwind CSS, JavaScript**, and powered by the **Google Gemini API**.

## ✨ Features
- 🎬 **YouTube Script & SEO Generator:** Auto-generates titles, scripts, tags, and descriptions.
- 📱 **Shorts/Reels Script Mode:** Fast-paced scripts with visual cues.
- 🎨 **AI Thumbnail Prompts:** Midjourney/DALL-E prompts for viral thumbnails.
- ⚡ **No Backend Required:** Runs 100% on the client-side (GitHub Pages Ready).
- 📱 **Fully Responsive:** Sleek dark-mode UI designed for mobile and desktop.

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Pure JavaScript (Async Fetch API)
- Google Gemini 1.5 Flash API

## 🚀 How to Run Locally
1. Clone or download this repository.
2. Open `index.html` directly in your web browser.
3. Enter your Gemini API key (Get a free key from [Google AI Studio](https://aistudio.google.com/)).
4. Type your topic and click **Generate**!
